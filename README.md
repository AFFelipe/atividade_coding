📌 Atividade Coding – Faculdade SENAC

Este repositório contém uma atividade de programação web desenvolvida como parte do curso da faculdade SENAC, utilizando HTML, CSS e JavaScript para consumir uma API externa e exibir dados de forma interativa.

📋 Sobre o Projeto

O objetivo da atividade é praticar conceitos de desenvolvimento web básico e consumo de APIs em JavaScript, usando uma interface simples de HTML e CSS.

✨ Neste projeto, ao digitar um número no campo de entrada, o usuário pode visualizar dados de um Pokémon através da PokéAPI, incluindo:

🆔 Número do Pokémon

🧬 Nome

⚔️ Tipagens

📏 Altura

⚖️ Peso

🛠️ Tecnologias Utilizadas
Tecnologia	Uso
HTML	Estrutura da página web
CSS	Estilização visual
JavaScript	Comunicação com a API e lógica da aplicação
🚀 Funcionalidades

✔ Entrada de número para consultar um Pokémon
✔ Requisição assíncrona para a PokéAPI
✔ Exibição dinâmica dos dados obtidos da API
✔ Tratamento de erro para entradas inválidas

⚙️ Como Usar

Clone o repositório:

git clone https://github.com/AFFelipe/atividade_coding.git


Navegue até a pasta do projeto:

cd atividade_coding


Abra o arquivo index.html no seu navegador web.

Digite um número de 1 a 898 (ou um número válido de Pokémon) no campo de entrada e veja os dados aparecerem na tela.
